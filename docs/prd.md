# PRD

## Features

- [x] Present welcome message

- [x] Navigation

- [x] Help command  
  Show available options (list, view, edit, create, delete, help)

- [x] List command  
  Display all tasks (id, title)

- [x] View command  
  Receive task ID and return:
  - id  
  - name  
  - description  
  - date  
  - duration  
  - assignee  

- [x] Create command  
  Show input fields for:
  - name  
  - description  
  - date  
  - duration  
  - assignee  

- [x] Update command  
  Allow updating a task by:
  - Showing current task data  
  - Providing input fields alongside existing values  

- [x] Delete command  
  Allow deleting a task by:
  - Showing list of tasks with IDs  
  - Letting user choose a task  
  - Displaying full task details before deletion  
  - Asking for confirmation  

- [ ] Persistence  
  The app should persist tasks
